# Zectrack
全球最大群眾募資平台 [Kickstarter](https://www.kickstarter.com/) 有 [Kicktraq](https://www.kicktraq.com/) 分析神器，那就來做一個台灣最大群眾募資平台[嘖嘖](https://www.zeczec.com/)的分析神器 [Zectrack](https://zectrack.herokuapp.com/) 吧！

## 功能
- 排行過去 1 小時集資金額
- 追蹤累積成長金額和每日成長金額
- 追蹤累積成長人數和每日成長人數
